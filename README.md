DriveEase Drivers is a landing page website built for a startup that provides temporary drivers for customers’ own vehicles.

The platform solves a real-world problem where people want to travel using their own car or bike but prefer not to drive. Instead of booking a taxi, users can hire a verified driver who comes to their location and drives their vehicle.

This project focuses on building a trust-driven and conversion-focused UI, where users can quickly understand the service and directly book a driver via WhatsApp without the need for a complex backend system.

Key Features:

* Clean and modern UI inspired by real-world products
* WhatsApp-based booking system with pre-filled message
* First ride offer highlight (₹199) to attract users
* Clear “How It Works” flow for better user understanding
* Trust & Safety section to improve credibility
* Fully responsive design (mobile-first approach)
* Floating WhatsApp CTA for higher conversion

Tech Stack:

* HTML
* CSS
* JavaScript

Goal of the Project:
To validate a startup idea quickly by creating a simple yet effective landing page that focuses on user trust and easy booking, without overengineering the system.

This project demonstrates my ability to design user-focused interfaces, structure real-world product flows, and build practical solutions instead of just static websites.
